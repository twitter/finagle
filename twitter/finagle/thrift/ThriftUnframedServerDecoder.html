ThriftUnframedServerDecoder

class ThriftUnframedServerDecoder extends ReplayingDecoder[VoidEnum] with ThriftServerDecoderHelper

Thrift unframed server decoder doesn't assume messages are framed.

The implementation repeated attempts to parse data until it succeeds. This is fine for small requests from well-behaved clients, but will impose a performance penality for large requests or requests from misbehaved clients (e.g., sending one byte at a time).
