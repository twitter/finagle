com.twitter.finagle.tracing

Trace

object Trace extends AnyRef

Trace maintains an interleaved stack of TraceIds and Tracers. The semantics are as follows: when reporting, we always report the topmost TraceId. That action is reported to all the Tracers that are _below_ that point in the stack.
